# Wisper

Wisper is a full-stack social media platform for professionals to find job and businesses to find right talent. Built with Typescript, Node.js, Express, PostgresQL and Prisma. It features user authentication, posting, commenting, liking, and more.

## Features

- User authentication with email and password
- User profiles with profile images and bio
- Posting with images, captions, and comments
- Commenting on posts
- Liking on posts
- User search
- User profile page
- Post feed with pagination
- Post details page with comments
- Post editing and deletion
- Profile image editing and deletion
- Comment editing and deletion
- User blocking

## Technologies Used

- Typescript
- Node.js
- Express.js
- PostgresQL
- Prisma
- AWS S3

## Installation

1. Clone the repository
2. Install dependencies with `npm install`
3. Set up environment variables in `.env` file
4. Start the server with `npm start`
